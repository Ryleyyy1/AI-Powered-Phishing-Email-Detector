# Data Directory

Letakkan dataset Anda di folder ini.

## Dataset yang direkomendasikan:

1. **Phishing Email Dataset** - dari Kaggle
2. **Enron Email Dataset** - untuk legitimate emails
3. **CEAS 2008 Dataset**

## Format yang diharapkan:

CSV dengan kolom:
- `email_text`: Isi email
- `label`: 0 (legitimate) atau 1 (phishing)
